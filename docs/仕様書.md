## 1. プロジェクト概要

### 1.1 目的

SEMI規格に準拠した SECS/GEM 通信のシミュレーション環境を提供する。  
半導体製造装置とホストコンピューター(PC)間の通信テストを、実機なしで検証できるツール。

### 1.2 適用規格

| 規格番号 | 規格名 | 内容 |
|----------|--------|------|
| SEMI E37 | HSMS (High-Speed Message Services) | TCP/IP 通信プロトコル |
| SEMI E5  | SECS-II | メッセージフォーマット |
| SEMI E30 | GEM (Generic Equipment Model) | 装置制御モデル |

### 1.3 準拠レベル

**GEM Full Compliant** (SEMI E30 全必須機能実装)

## 4. GEM 状態機械

### 4.1 通信状態 (Communication State Model)

```
DISABLED
  │ HSMS接続確立
  ▼
WAIT_CR_FROM_HOST ──T7タイムアウト──► WAIT_DELAY
  │ S1F13受信
  ▼
COMMUNICATING
  │ HSMS切断
  ▼
DISABLED
```

| 状態 | 値 | 説明 |
|------|----|------|
| Disabled | 0 | 通信無効 |
| WaitCrFromHost | 1 | S1F13 受信待ち |
| WaitDelay | 2 | T7 タイムアウト後の遅延待ち |
| WaitCr | 3 | CR 送信待ち |
| Communicating | 4 | 通信中 |

### 4.2 制御状態 (Control State Model)

```
EQUIPMENT_OFFLINE
  │ S1F13 (通信確立)
  ▼
ATTEMPT_ONLINE
  │ S1F17 (ON-LINE 要求)
  ▼
ONLINE_REMOTE ◄──S2F41 LOCAL──► ONLINE_LOCAL
  │ S1F15 (OFF-LINE 要求)
  ▼
HOST_OFFLINE
```

| 状態 | 値 | 説明 |
|------|----|------|
| EquipmentOffline | 0 | 装置オフライン |
| AttemptOnline | 1 | オンライン試行中 |
| HostOffline | 2 | ホスト主導オフライン |
| OnlineLocal | 3 | オンライン / ローカル制御 |
| OnlineRemote | 4 | オンライン / リモート制御 |

### 4.3 処理状態 (Processing State Model)

```
INIT → IDLE ⇄ SETUP → READY → EXECUTING ⇄ PAUSE
                                    │
                                  IDLE (完了)
```

| 状態 | 値 | 説明 |
|------|----|------|
| Init | 0 | 初期化中 |
| Idle | 1 | アイドル |
| Setup | 2 | セットアップ中 |
| Ready | 3 | 実行準備完了 |
| Executing | 4 | 実行中 |
| Pause | 5 | 一時停止中 |

## 7. リモートコマンド (S2F41)

| コマンド名 | 動作 | 前提条件 |
|-----------|------|---------|
| START | 処理状態を Executing に遷移 | READY または IDLE |
| STOP | 処理状態を Idle に遷移 | EXECUTING または PAUSE |
| PAUSE | 処理状態を Pause に遷移 | EXECUTING |
| RESUME | 処理状態を Executing に遷移 | PAUSE |
| LOCAL | 制御状態を OnlineLocal に遷移 | オンライン中 |
| REMOTE | 制御状態を OnlineRemote に遷移 | オンライン中 |

**HCACK コード:**

| 値 | 意味 |
|----|------|
| 0 | ACK (コマンド受け付け) |
| 1 | 未知のコマンド |
| 3 | コマンド実行不可 (状態不正) |
